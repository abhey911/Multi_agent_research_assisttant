# Outputs Directory

This directory contains generated research reports.

## Structure

- `reports/` - Contains all research reports in markdown format

## Report Naming

Reports are automatically named based on:
- Research topic (cleaned and shortened)
- Timestamp (YYYYMMDD_HHMMSS)

Example: `quantum_computing_cryptography_20251201_143022.md`

## Note

This directory is created automatically when the first report is generated.
